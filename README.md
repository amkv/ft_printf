# ft_printf
my implementation of printf function on C language

1. flags 
```
+ - # 'space' 0
```
2. width
3. precision
4. length
```
h hh l ll j z
```
5. modifiers
```
d D i c C s S u U x X o O p
```
6. (* 'star') arguments

how to check?
compile, launch and test :)
```
make re
gcc -Wall -Wextra -Werror temp/main.c libftprintf.a
./a.out
```

or launch run.sh script
```
sh run.sh
```
